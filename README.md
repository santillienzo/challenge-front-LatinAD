## Ejecución
### 1) Crear un archivo .env con el siguiente contenido: 
```
VITE_URI=https://challenge-front-7fw1.onrender.com
```

### 2) Instalar las dependencias
```
npm install
```

### 3) Ejecutar el proyecto
```
npm run dev
```

## Tecnologías y lenguajes utilizados
 * React + Vite
 * Typescript
